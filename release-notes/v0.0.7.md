## v0.0.7 - SMS Webhooks & Dynamic Version

### Features

- **New `sip caller-id` command** - Set caller ID for SIP accounts:
  ```
  zadarma-cli sip caller-id --id <ID> --number <NUMBER>
  ```
- **SMS webhook management** - Added `sms get-webhook` and `sms set-webhook` commands
- **SMS listen with auto-validation** - Added `--webhook` flag to automatically register webhook and wait for Zadarma's `zd_echo` validation
- **Improved statistics output** - Added more columns: SIP, From, To, Duration, Cost (with billed cost), Currency, Description, Disposition, Hangup cause
- **Dynamic version** - Version is now injected from git tag at build time

### Improvements

- Restructured phone commands: `phone countries`, `phone country <ISO>`, `phone number <NUM>`
- Fixed statistics JSON parsing (handle both `data` and `stats` response fields)
- Fixed lint errors across codebase
- Added AGENTS.md for AI agent rules
- Documentation updates (README, SKILL)
- Added pre-commit hooks (gofmt, golangci-lint, go test)

### Cleanup

- Removed deprecated `webhook` command (moved under `sms`)
- Anonymized sample data for public release
- Added `.ai/` to .gitignore
