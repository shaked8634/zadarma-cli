## v0.0.3 - Webhooks & Debug Mode

### New Features

- **Webhook commands** — Manage Zadarma webhooks:
  - `webhook set <url>` — Set notification URL
  - `webhook get` — Get current webhook URL  
  - `webhook listen` — Local SMS listener server
- **Debug mode** — Use `-d` or `--debug` to trace API requests

### Improvements

- Simplified README with practical use cases
- Documented release conventions in NOTES.md
